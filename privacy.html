Política de Privacidade

A tua privacidade é levada a sério. Aqui está, com todas as letras, o que fazemos com os teus dados.

Navegação rápida

01Introdução

A Playzup é uma aplicação móvel destinada à gestão de equipas de futebol amador. Esta política descreve como recolhemos, utilizamos e protegemos os teus dados pessoais. Para qualquer questão relacionada com privacidade, podes contactar-nos através de suporte@playzup.app.

02Dados recolhidos

Para que a app funcione corretamente, podemos recolher os seguintes dados:

  • Nome e endereço de email (para criação de conta e identificação na equipa)
  • Dados de jogo (presenças, golos, assistências, votações de MVP, estatísticas)
  • Dados de pagamento processados por terceiros (não armazenamos diretamente dados de cartão)
  • Identificadores técnicos do dispositivo necessários ao funcionamento da app

03Como usamos os teus dados

Utilizamos os dados recolhidos para:

  • Permitir o funcionamento das funcionalidades da app (convocatórias, equipas, estatísticas)
  • Melhorar a experiência e desenvolver novas funcionalidades
  • Enviar comunicações essenciais relacionadas com a tua conta ou jogos

04Partilha de dados

Não vendemos os teus dados a terceiros. Para que a app funcione, partilhamos dados estritamente necessários com os seguintes serviços:

  • RevenueCat — processamento de subscrições e pagamentos in-app
  • Firebase (Google) — autenticação, base de dados e infraestrutura

05Segurança

Utilizamos o Firebase Authentication para gerir o acesso à tua conta e o Firestore com regras de segurança configuradas para que apenas tu e os membros autorizados do teu clube possam aceder aos respetivos dados. Aplicamos boas práticas de segurança para proteger a informação contra acessos não autorizados.

06Direitos do utilizador

Tens o direito de:

  • Aceder aos dados pessoais que detemos sobre ti
  • Solicitar a correção de dados incorretos ou desatualizados
  • Solicitar a eliminação da tua conta e dos dados associados

Para exercer qualquer destes direitos, basta enviar um email para suporte@playzup.app.

07Cookies e analytics

Utilizamos serviços de analytics de forma mínima, exclusivamente para compreender como a app é utilizada e para a melhorar. Não recolhemos dados para fins publicitários nem partilhamos com plataformas de marketing.

08Alterações a esta política

Esta política pode ser atualizada periodicamente. Sempre que existirem alterações relevantes, serás notificado através da própria app.
